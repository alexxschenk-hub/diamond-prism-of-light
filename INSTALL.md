# DIAMOND – Prism of Light

**"Don't just see the news. See the spectrum."**

## Installation in Chrome

1. Offne Chrome und gehe zu `chrome://extensions/`
2. Aktiviere oben rechts den **Entwicklermodus** (Developer Mode)
3. Klicke auf **"Entpackte Erweiterung laden"** (Load unpacked)
4. Wahle diesen `diamond-extension` Ordner aus
5. Das Diamond-Icon erscheint in deiner Browser-Toolbar

## Benutzung

1. Offne eine Nachrichtenseite (z.B. orf.at, spiegel.de, bbc.com)
2. Klicke auf das Diamond-Icon in der Toolbar
3. Schalte **"Analyse aktiv"** ein
4. Die Seite wird analysiert und manipulative Muster farbig markiert

## Die Farbkanale

| Farbe | Kanal | Erkennt |
|-------|-------|---------|
| Rot | Infrarot | Angst & Bedrohung |
| Orange | Dringlichkeit | Kunstlicher Zeitdruck |
| Gelb | Profit | Finanzielle Interessen |
| Grun | Wir vs. Die | Kunstlicher Gruppenzwang |
| Blau | Autoritat | Vage Quellen |
| Violett | Ultraviolett | Moralische Uberhohung |

## Die drei Schliffe

- **Rohdiamant**: Farbige Markierungen im Text + Sidebar-Analyse
- **Brillant**: Entfernt manipulative Adjektive, zeigt nur die Fakten
- **Feuer**: (Kommt bald) Zeigt Gegenperspektiven
